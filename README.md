# BharatKala – Traditional Indian Art & Heritage Marketplace (Template)**

BharatKala is a culture-inspired website template crafted to showcase India’s traditional arts, crafts, textiles, fragrances, and handmade creations. Adapted from the Ecothunder free template, it transforms the original layout into a warm, elegant, heritage-focused design suitable for artisan marketplaces or cultural showcases.
# Features

# Elegant Front Page
* Custom BharatKala logo
* Hero introduction section
* Showcase images of Channapatna toys, Mitti perfume, and Odisha handloom sarees

# Structured Pages
* Home
* About Us
* Why Choose Us
* Contact
* Blog (includes 3 crafted cultural articles)

# Responsive Navigation
* Clean, aligned navbar
* Updated menu styling
* Fixed layout issues from the base template

# Footer Section
* Brand summary
* Social media links
* Newsletter input
